# Hesap Makinesi Projesi

## Proje Açıklaması
Bu proje Python kullanılarak geliştirilmiş basit bir hesap makinesidir.

## Özellikler
- Toplama
- Çıkarma
- Çarpma
- Bölme

## Kullanım
1. main.py çalıştırılır
2. Menüden işlem seçilir
3. Sayılar girilir

## Dosya Yapısı
- main.py → program akışı
- logics.py → matematik işlemleri
- interface.py → kullanıcıdan veri alma ve gösterme
